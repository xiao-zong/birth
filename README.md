# Birth-death processes
Aihua 想要的结果是对于一个a birth-death process with multiple births and single death, the process itself has a unique stationary distribution when conditioning on the total number of the process having unique stationary distribution 

under minimum condition: the smallest death rate is greater than the largest birth rate

Let the initial point process be a Gibbs point process.

When the total number of the point process is ergodic, the whole process is ergodic.

To ensure unique a stationary distribution, well-pose martingale problem is needed.
